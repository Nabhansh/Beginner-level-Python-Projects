## 🚀 How to Run

```bash

python 01_calculator.py

# ... etc
```

## ⚙️ Special Setup

### Project 11 - Weather App
1. Get a **free** API key from [openweathermap.org](https://openweathermap.org/api)
2. Open `11_weather_app.py`
3. Replace `YOUR_API_KEY_HERE` with your key

### Project 13 - YouTube Downloader
```bash
pip install yt-dlp
# For MP3 download, also install FFmpeg: https://ffmpeg.org
```

## 🎯 Features Highlights

- **Tic-Tac-Toe**: Score tracking, win highlighting, theme toggle
- **Digital Clock**: Live seconds bar, alarm with notification
- **Password Generator**: Strength checker, bulk generation
- **Calculator GUI**: Full scientific functions, keyboard support
- **Task Planner**: Persistent JSON storage, priorities, categories
- **File Rename Tool**: 12 rename operations including regex
- **Email Slicer**: Bulk analysis, provider stats, JSON export
- **Unit Converter**: 7 categories (length, weight, temp, area, speed, volume, time)

## 📚 Requirements
- Python 3.7 or higher
